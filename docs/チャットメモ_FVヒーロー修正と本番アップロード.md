## 1. FV ヒーローまわりの修正

### 経緯
- キャッチコピー「保護犬・猫に囲まれて、自分らしく働く。」の下にある  
  「保護犬猫カフェ×就労継続支援B型」「川崎駅徒歩圏内」「時給換算400円〜」が  
  文字が小さく目立たないという要望があった。

### やったこと（段階的に）

1. **まず「わかりやすく」する**
   - 3つの情報を白背景＋茶色の枠の角丸ボックスでひとまとまりにした。
   - 文字を大きく（SP: text-sm font-bold、PC: text-base font-semibold）。
   - 「400円〜」だけオレンジ（#E86833）＋太字で強調。
   - アイコン（MapPin, Coins）をオレンジで表示。
   - 「時給400円〜」→「時給換算400円〜」に表記統一。

2. **⚪で囲む形に変更**
   - ユーザー要望: 各項目を丸で囲み、  
     [犬アイコン] 動物×就労支援  
     [マップアイコン] 川崎駅徒歩圏内  
     [お金アイコン] 時給換算400円〜  
   - 1行目を「保護犬猫カフェ×就労継続支援B型」→「**動物×就労支援**」に変更。
   - `lucide-react` の `Dog` アイコンを追加。
   - 各項目を `rounded-full` のピル型バッジ（白背景・茶枠）で囲み、縦並びで表示。

3. **横並び・真円に変更**
   - 添付画像を参考に「横並び」「真円で囲む」形に変更。
   - **横並び:** `flex` で 3 つを横一列に配置（SP/PC とも）。SP では `flex-wrap` で折り返し可能に。
   - **真円:** 各ブロックを `h-20 w-20`（SP）／`h-24 w-24`（PC）の同じ縦横で `rounded-full` にし、正円の枠で表示。
   - 円の中は上にアイコン、下にテキストを中央揃え。「時給換算400円〜」は 2 行（時給換算 / 400円〜）で収めた。

### 変更したファイル
- `components/hero-section.tsx`
  - import に `Dog` を追加。
  - SP/PC それぞれで、上記の「横並び・真円」のブロックを実装。

## 2. 本番への反映（Git とドキュメント）

### やったこと
- 上記ヒーロー修正を含む変更を `git add` → `git commit` → `git push`（`origin main`）した。
- コミットメッセージ: `Hero FV circles and docs for production deploy`
- **注意:** Cursor 内蔵ターミナルでは `git commit` に `--trailer` が付いて失敗したため、`/usr/bin/git commit` で実行した。

### 本番で反映されなかった理由の整理
- 本番の LP は **Xサーバー**（a-melo.com/anella-work-b/）の静的ファイルで表示されている。
- **Git push** で更新されるのは **Vercel**（フォーム API）のみ。Xサーバー上のファイルは更新されない。
- そのため「LP の見た目」を変えたときは、**ビルド（2.2）** と **Xサーバーへのアップロード（2.3）** が必須。

## 3. ドキュメントの更新（本番アップロードの流れ）

### 更新したファイル

### 追加・修正した内容

   - ❌ `anella-work-b` の横に `out` ができる → ✅ `anella-work-b/index.html` など中身が直接 `anella-work-b` に入る。

3. **「今回のようにファーストビューを直した場合」の手順**
   - 2.2 のビルドコマンド（1行）を記載。
   - 2.3 で「`out` の中身を `public_html/anella-work-b` に上書き」と明記。
